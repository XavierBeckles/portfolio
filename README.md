# Portfolio

Personal portfolio site, built with Next.js and Tailwind CSS.

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## To customize

- `src/app/page.tsx` — content lives in one file: hero copy, the `links` object
  (update the LinkedIn URL), the featured project card, and the `skills` list.
- Add more project cards by copying the `<article>` block inside `Projects()`.
- `src/app/globals.css` — theme colors (`--accent`, `--background`, etc.).

## Deploy

Deploys cleanly to [Vercel](https://vercel.com/new) — import this repo and it
will pick up the Next.js build automatically.
